# IMDb Movies Analysis using SQL

## Project Overview
This project analyzes the IMDb Movies Database using SQL to extract meaningful insights
to help Bolly Movies plan future movie projects.

## Objectives
- Analyze movie release trends
- Identify popular genres
- Evaluate movie ratings and votes
- Analyze directors, actors, and production houses
- Provide data-driven recommendations

## Tools Used
- SQL (MySQL / PostgreSQL compatible)
- IMDb Dataset

## Repository Contents
- IMDb_Movies_Analysis_SQL.sql : Complete SQL queries for all analysis segments
- IMDb_Movies_Analysis_Report.docx : Analysis insights and final recommendations

## Conclusion
Based on the analysis, Thriller and Drama genres with high-rated crew members
and early-year releases have higher success potential.
